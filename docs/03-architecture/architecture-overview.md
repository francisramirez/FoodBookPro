# Architecture Overview

- Clean Architecture
- Modular monolith
- API-first
- Event-driven opcional para notificaciones

## Capas
- API
- Application
- Domain
- Infrastructure
- Worker

- API de reservas
- Servicio de disponibilidad
- Módulo de clientes
- Persistencia relacional
- Servicio de notificaciones
- Telemetría y logging

- Facilidad de pruebas
- Escalabilidad
- Trazabilidad
- Mantenibilidad
