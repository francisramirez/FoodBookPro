# SRS - FoodBook Pro

## 1. Introducción
FoodBook Pro es un sistema orientado a reservas para restaurantes.

## 2. Alcance
El sistema permitirá registrar restaurantes, mesas, clientes y reservas; además podrá notificar confirmaciones y cancelaciones.

## 3. Requisitos funcionales
- RF-001 Registrar restaurantes
- RF-002 Registrar mesas
- RF-003 Consultar disponibilidad
- RF-004 Crear reservas
- RF-005 Confirmar reservas
- RF-006 Cancelar reservas
- RF-007 Gestionar clientes
- RF-008 Consultar historial de reservas

## 4. Requisitos no funcionales
- RNF-001 Seguridad
- RNF-002 Rendimiento
- RNF-003 Disponibilidad
- RNF-004 Escalabilidad
- RNF-005 Observabilidad
- RNF-006 Auditabilidad
