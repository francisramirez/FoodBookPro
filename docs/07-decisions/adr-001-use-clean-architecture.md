# ADR-001 - Use Clean Architecture

## Estado
Aceptado

## Contexto
Se requiere una base mantenible, desacoplada y apta para crecimiento.

## Decisión
Adoptar Clean Architecture con capas API, Application, Domain e Infrastructure.

## Consecuencias
- Mejor mantenibilidad
- Mejor capacidad de pruebas
- Mayor claridad de responsabilidades
